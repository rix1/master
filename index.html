Wireless ECG:

Repository overview

The following is an overview of the different repositories related to my master thesis. The project for the thesis itself, can be found here.

  • Monitoring Server

    Meteor server for receiving data from BLE devices and publishing it to clients in real time.

    Project: Meteor
    License: MIT
    Copyright (C) 2011 – 2016 Meteor Development Group

  • Time Synchronizing Server

    Node server for synchronizing time between clients. Using the following open source software:

    Project: TimeSync
    License: MIT
    Copyright (c) 2016 enmasse.io

  • BLE Emit Logger

    Node server for registering when a nRF51 board emits data over BLE.

    Project: TimeSync
    License: MIT
    Copyright (c) 2016 enmasse.io

    Project: node-serialport
    License: MIT
    Copyright 2010, 2011, 2012 Christopher Williams

    Project: node-ddp-client
    License: MIT/X11
    Copyright (C) 2013 – 2016 by Tom Coleman

  • Android Sensor Gateway

    An Android gateway for proxying data from BLE devices to Meteor servers.

    Project: Android-DDP
    License: Apache 2.0
    Copyright (c) delight.im

  • Sensor Software

    mBed software for Nordic Semiconductor nRF51822 Development kit. Used in our end-to-end latency experiment.

    Project: ARM mbed Microcontroller Library
    License: Apache 2.0
    Copyright (c) 2006-2015 ARM Limited
